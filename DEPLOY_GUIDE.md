## 1. Деплой на Vercel

**Vercel** — бесплатный сервис для публикации сайтов. Он автоматически обновляет сайт каждый раз, когда ты вносишь изменения в код.

### Шаг 1 — Создай аккаунт на Vercel

1. Перейди на сайт **[vercel.com](https://vercel.com)**
2. Нажми кнопку **"Sign Up"**
3. Выбери **"Continue with GitHub"** — войди через свой GitHub аккаунт
4. Дай Vercel доступ к своим репозиториям

### Шаг 2 — Подключи репозиторий

1. На главной странице Vercel нажми кнопку **"Add New…"** → **"Project"**
2. В списке найди репозиторий **MDK-TRANSPORT** и нажми **"Import"**
3. На следующей странице ничего не меняй — всё уже настроено
4. Нажми большую кнопку **"Deploy"**
5. Подожди 1–2 минуты — Vercel соберёт и опубликует сайт

После этого Vercel выдаст тебе адрес вида `mdk-transport-xxx.vercel.app` — это твой сайт в интернете.

> **Важно:** При каждом новом коммите в ветку `main` сайт будет обновляться автоматически.

## 2. Подключение своего домена

Если у тебя есть домен (например, `mdktransport.com`), подключи его к Vercel.

### Шаг 1 — Добавь домен в Vercel

1. Открой свой проект на Vercel
2. Перейди во вкладку **"Settings"** → **"Domains"**
3. Введи свой домен (например, `mdktransport.com`) и нажми **"Add"**
4. Vercel покажет тебе DNS-записи, которые нужно добавить

### Шаг 2 — Настрой DNS у регистратора домена

Зайди в панель управления там, где ты купил домен (GoDaddy, Namecheap, reg.ru и т.д.) и добавь указанные Vercel записи:

**Вариант А — если хочешь использовать корневой домен (`mdktransport.com`):**
| Тип | Имя | Значение |
|-----|-----|---------|
| `A` | `@` | `76.76.21.21` |

**Вариант Б — если хочешь использовать `www.mdktransport.com`:**
| Тип | Имя | Значение |
|-----|-----|---------|
| `CNAME` | `www` | `cname.vercel-dns.com` |

> DNS может обновляться до 24 часов. Обычно занимает 10–30 минут.

### Шаг 3 — Добавь URL сайта в переменные окружения

После подключения домена обязательно выполни следующий шаг.

## 3. Переменные окружения

Переменные окружения — это настройки, которые Vercel передаёт твоему сайту. Без них некоторые функции (sitemap, Google верификация) не будут работать правильно.

### Как добавить переменные

1. Открой проект на Vercel
2. Перейди в **"Settings"** → **"Environment Variables"**
3. Для каждой переменной: введи имя в поле **"Key"**, значение в поле **"Value"**, нажми **"Save"**

### Необходимые переменные

#### `NEXT_PUBLIC_SITE_URL` — **обязательная**

| Поле | Значение |
|------|---------|
| Key | `NEXT_PUBLIC_SITE_URL` |
| Value | `https://mdktransport.com` *(или твой реальный домен)* |

Эта переменная нужна для корректного формирования `sitemap.xml` и `robots.txt`.

#### `NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION` — нужна для Google

| Поле | Значение |
|------|---------|
| Key | `NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION` |
| Value | *(код из Google Search Console — см. раздел 4)* |

После добавления переменных нажми **"Redeploy"** в разделе Deployments, чтобы применить изменения.

## 4. Google Search Console

**Google Search Console** — бесплатный инструмент от Google, который позволяет добавить сайт в поиск и следить за его позициями.

### Шаг 1 — Открой Google Search Console

1. Перейди на **[search.google.com/search-console](https://search.google.com/search-console)**
2. Войди через свой Google аккаунт
3. Нажми **"Добавить ресурс"** (или **"Add property"**)

### Шаг 2 — Добавь свой сайт

1. Выбери **"Домен"** если хочешь охватить все поддомены, или **"URL-префикс"** для конкретного адреса
2. **Рекомендуем:** выбери **"URL-префикс"** и введи `https://mdktransport.com`
3. Нажми **"Продолжить"**

### Шаг 3 — Подтверди владение сайтом (метатег)

Google предложит несколько способов. Выбери **"Тег HTML"**:

1. Google покажет тебе тег вида:
   ```
   <meta name="google-site-verification" content="AbCdEfGhIjKlMnOp123456" />
   ```
2. Скопируй только значение content — строку `AbCdEfGhIjKlMnOp123456` (у тебя будет другое значение)
3. Вставь это значение в переменную окружения `NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION` на Vercel (см. раздел 3)
4. Сделай Redeploy на Vercel
5. Вернись в Google Search Console и нажми **"Подтвердить"**

> Если Vercel уже задеплоил обновлённую версию, Google найдёт метатег и подтвердит владение.

## 5. Отправка карты сайта (Sitemap)

После подтверждения сайта нужно сообщить Google о структуре сайта, чтобы он быстрее его проиндексировал.

### Как отправить Sitemap

1. В Google Search Console перейди в раздел **"Индексирование"** → **"Файлы Sitemap"**
2. В поле **"Добавить новый файл Sitemap"** введи:
   ```
   sitemap.xml
   ```
3. Нажми **"Отправить"**

Google начнёт обходить сайт. Обычно это занимает от нескольких часов до нескольких дней.

## 6. Проверка

После всех настроек убедись, что всё работает:

### Проверь sitemap

Открой в браузере: `https://твой-домен.com/sitemap.xml`

Должен открыться XML-файл с адресом твоего сайта.

### Проверь robots.txt

Открой в браузере: `https://твой-домен.com/robots.txt`

Должно быть примерно так:
```
User-agent: *
Allow: /

Sitemap: https://твой-домен.com/sitemap.xml
```

### Проверь поиск на сайте

На самом сайте в шапке появилась кнопка с иконкой лупы. Нажми её (или нажми **Ctrl+K** на клавиатуре) и попробуй что-нибудь найти, например: «страховка» или «депозит».

### Проверь SEO-теги

1. Открой сайт в браузере
2. Нажми правой кнопкой мыши → **"Просмотр кода страницы"** (или **Ctrl+U**)
3. Найди в коде строки с `og:title`, `twitter:card`, `application/ld+json` — они должны присутствовать
